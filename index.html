Инструкция по установке клиентского сайта на сервер:

  1. Скопировать все папки и файлы с гит хаба себе на сервер.
  2. Папка yii находиться вне корневой папке сайта public_html. 3.Необходимо создать новую базу и пользователя на хостинге и импортировать в базу gclick.sql

  3. Прописать название новой базы, пользователя и пароль в конфигурационном файле protected/config/main.php В строках 'db'=>array( 'connectionString' => 'mysql:host=localhost;dbname=Название Вашей базы', 'emulatePrepare' => true, 'username' => 'Имя пользователя', 'password' => 'Пароль', 'charset' => 'utf8', 'tablePrefix'=>'pok_', ),

  4. your domen/site/login - ссылка для авторизации на сайте, admin admin - логин и пароль по-умолчанию.

  5. Для того, чтобы в админке появилась возможность импортировать купоны необходимо прописать Partner ID из партнерской системы primelead. 7.После того как в партнерской системе прописан Partner ID, появится возможность импортировать купоны.
